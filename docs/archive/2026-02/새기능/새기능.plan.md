# Plan: 애니메이션 효과 (새기능)

## 개요

| 항목 | 내용 |
|------|------|
| 기능명 | 애니메이션 효과 |
| 우선순위 | Medium |
| 예상 난이도 | Medium |
| 작성일 | 2026-02-21 |

## 배경 / 목적

ISTJ 포트폴리오에 스크롤 기반 인터랙션 및 진입 애니메이션을 추가하여 사용자 경험을 향상시킵니다.
정적인 포트폴리오를 생동감 있게 만들고, 방문자의 시선을 자연스럽게 유도하는 것이 목적입니다.

## 요구사항

### 기능 요구사항
- [ ] 섹션 진입 시 Fade-in 애니메이션 (스크롤 감지)
- [ ] 프로젝트 카드 Hover 효과 (scale, shadow)
- [ ] 헤더 스크롤 시 배경 blur/opacity 전환
- [ ] SkillBar 진입 시 width 애니메이션 (0 → 실제값)
- [ ] 페이지 초기 로드 시 Hero 섹션 등장 애니메이션

### 비기능 요구사항
- [ ] `prefers-reduced-motion` 미디어 쿼리 준수 (접근성)
- [ ] 애니메이션 성능 최적화 (will-change, GPU 가속)
- [ ] ISTJ 미니멀 정체성 유지 (과하지 않은 효과)

## 기술 접근

### 방식: Framer Motion + CSS Transitions
- `framer-motion` 라이브러리 활용 (React 기반 선언적 애니메이션)
- Intersection Observer API로 스크롤 감지
- Tailwind CSS `transition-*` 유틸리티 병행 활용

### 애니메이션 설계
| 컴포넌트 | 효과 | 트리거 |
|----------|------|--------|
| Hero 섹션 | slideUp + fadeIn | 페이지 로드 |
| 섹션 공통 | fadeIn | 스크롤 진입 |
| ProjectCard | scale(1.02) + shadow | hover |
| SkillBar | width 0→value | 스크롤 진입 |
| Header | backdrop-blur 강화 | 스크롤 |

## 구현 범위

### In Scope
- Framer Motion 설치 및 설정
- 섹션별 진입 애니메이션 (Hero, About, Projects, Skills)
- 카드 hover 인터랙션
- SkillBar fill 애니메이션
- Header 스크롤 효과 강화

### Out of Scope
- 페이지 전환 애니메이션 (route transition)
- 마우스 파티클, 커서 커스터마이징
- 3D 효과 (Three.js 등)

## 파일 영향 범위

```
package.json                    ← framer-motion 추가
components/
  atoms/SkillBar.tsx            ← width 애니메이션
  molecules/ProjectCard.tsx     ← hover 효과
  organisms/Header.tsx          ← 스크롤 blur 효과
  templates/PortfolioLayout.tsx ← 섹션 fadeIn wrapper
lib/
  hooks/useInView.ts            ← Intersection Observer 훅 (신규)
```
