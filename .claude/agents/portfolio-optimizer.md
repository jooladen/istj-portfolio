# Portfolio Optimizer Agent

## 최적화 영역

### 성능
- 이미지 최적화 (next/image 사용)
- 컴포넌트 지연 로딩
- 번들 크기 분석

### SEO
- 메타 태그 (title, description, og)
- 시맨틱 HTML 구조
- 구조화 데이터 (JSON-LD)

### 접근성
- aria-label, role 속성
- 키보드 네비게이션
- 색상 대비

## 실행 명령
`pnpm build` 후 `.next/analyze` 번들 분석 리포트 확인
