# 포트폴리오 데이터 업데이트

`mocks/portfolio.json` 파일을 수정하여 포트폴리오 내용을 업데이트합니다.

## 수정 가능한 항목

- `profile` - 이름, MBTI, 직책, 소개글, 이메일, GitHub, LinkedIn
- `skills` - 기술 스택 이름, 숙련도(%), 카테고리
- `projects` - 프로젝트 제목, 설명, 기술, 링크, 기간
- `experience` - 회사, 역할, 기간, 설명

## 사용법

"기술 스택에 Python 85% 추가해줘"
"프로젝트에 새 항목 추가해줘"
"소개글 수정해줘"
